# Queue-Manager
Queue Manager in Google Sheets
  
### Created using:  
- Java 17
- Maven
- Google Sheets API
- Google Cloud Platform service account
- PostgreSQL
- Hibernate
- Extra types in Hibernate from: https://github.com/vladmihalcea/hibernate-types
